call apply and bind function

.call() method is used to call a function for object and when passing other argument in function we simply pass argument using comma.

apply() method is also used to call the function and when passing more argument we need to create arraylike list.

.bind() method makes copy of function but not directly invoke the function. After binding the function the function could be invoked.

.bind() method create new function of called function and we can invoke it easily.
