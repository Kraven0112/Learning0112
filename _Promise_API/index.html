Promise API in javascript.

Promise.all() =>In this case if all the promises are resolved then all the promises are fulfilled. If one of the promises gets rejected then it will return the error.

Promise.allSettled() => return array of promise if all are resolved . Also give the array of promises even promises get rejected

Promise.race() => This promise method return single promise as a value which can be resolved first. It depends on the time of promise which can be resolved first.

Promise.any() => This method is success seeker api. Returns single promise. If all promises are rejected it gives aggregated error. Aggregate error means array of errors. It waits for first Settled success.

Settled means resolve as well as reject, success as well as failure and fulfilled as wel as rejected.
